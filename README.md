# Telegram Web Renderer Bot

A lightweight Telegram bot that captures full-page screenshots or generates PDFs of websites and sends them directly to the chat. All traffic goes through Telegram only. Built to run entirely inside Docker.

---

## 🚀 Features

- 📸 Capture full-page screenshots of webpages
- 📄 Generate styled PDFs of pages
- 🔒 User access restricted via Telegram ID whitelist
- 🐳 Docker-only deployment

---
